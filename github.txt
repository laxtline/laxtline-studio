6) LIVE karne ke STEPS (GitHub Pages)
------------------------------------------------------------
  1. github.com par login (username: suryafx013).
  2. "New repository" -> naam upar wala (Option A ya B) ->
     Public -> Create.
  3. Repo me poora project upload karo:
        index.html   (ye hi main/home page khulega)
        css/  folder  (poora)
        js/   folder  (poora)
        Logo  & Background/  folder  (poora)
        baki sab files (robots.txt, sitemap.xml, site.webmanifest...)
     -> Drag & drop ya "Add file > Upload files".
  4. Repo -> Settings -> Pages ->
        Source: "Deploy from a branch"
        Branch: main  /  (root)  -> Save.
  5. 1-2 minute wait karo -> upar Pages section me live URL aa jayega.
  6. URL kholo, ek baar HARD REFRESH (Ctrl + F5).


------------------------------------------------------------
7) DEPLOY ke baad DHYAN DENE ki baatein
------------------------------------------------------------
  - GitHub by default "index.html" hi serve karta hai (home page).
    "SURYA_FX.com.html" sirf backup hai — usse deploy mat karo.
  - GitHub case-sensitive hai. Folder/file ke naam BILKUL waise hi
    upload ho jaise project me hain, especially:
        "Logo  & Background"   (beech me DO space hai — dhyan se)
    warna logo/background image 404 ho sakta hai.
  - Saari project videos Cloudinary se aati hain (internet se),
    is liye unhe upload karne ki zaroorat nahi — apne aap load hongi.
  - Har update ke baad live site par Ctrl + F5 (cache clear) karo.
