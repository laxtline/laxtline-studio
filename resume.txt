  2) SHORT SUMMARY  (2-3 lines — ideal for a resume Projects section)

  Designed and developed LAXTLINE, a responsive single-page portfolio website
  for a video editor, featuring an animated hero, custom cursor and an
  interactive video gallery with a fullscreen viewer. Built with HTML5, CSS3 and
  Vanilla JavaScript, and engineered a lazy-loading + memory-management system to
  smoothly handle 50+ videos without performance loss.


--------------------------------------------------------------------------------
  3) RESUME BULLET POINTS  (drop straight under the project name)
--------------------------------------------------------------------------------

  • Built a responsive, single-page portfolio website from scratch using HTML5,
    CSS3 and Vanilla JavaScript (no frameworks).
  • Developed an interactive video gallery with hover-to-play previews and a
    fullscreen viewer offering play/pause, seek, volume and keyboard controls.
  • Engineered a lazy-loading and memory-release system that keeps the page fast
    and crash-free while displaying 50+ videos (~600 MB of media).
  • Implemented smooth UI animations (canvas hero, custom cursor, scroll reveals)
    while maintaining performance via GPU-friendly CSS and throttled events.
  • Added SEO optimisation (meta tags, Open Graph, sitemap, robots.txt) and a PWA
    web manifest for better reach and shareability.
  • Refactored a single-file codebase into a clean, documented HTML/CSS/JS
    structure for maintainability and scalability.
